batsman: static site generator

batsman aims to be a zero-configuration static site generator. The README has a quick start section and more. The source code is on GitHub (MIT License).

You can install batsman by running:

go get -u github.com/nishanths/batsman

batsman automatically minifies generated content, has an HTTP server to serve generated content, and can regenerate content on change.

If you got to this page by running batsman serve locally, you can find the site source in src and the generated site in build.
